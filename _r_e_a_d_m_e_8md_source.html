1 # MoGUL
2 MoGUL - Mosaic Game Utility Library. A suite of libraries designed to facilitate game programming. Taken as a whole it will be a comprehensive game engine library.

10 <H2>Libraries</H2>
11 MoGUL is a collection of libraries that can be used as a whole or taken in parts. The libraries are broken down into libraries that can be used independently from the rest of MoGUL, the core set of game related libraries and optional extension libraries.
12 <H3>Stand Alone</H3>
13 These libraries provide useful utility for the C language that the rest of MoGUL will use, but these are not dependent on the rest of MoGUL.
14  - mgl_types - [complete] - a library of type aliases. Mostly bult from the SDL standard types, but also provides some common types that will be useful in games in general such as vectors (the math construct, not the linked list) and matrices. mgl_dict is a complex generic type that will work well storing config data loaded from JSON, XML or YAML
15  - mgl_dict - [complete] - a dictionary type primary used for supporting config
16  - mgl_text - [complete] - support for finite length strings. Used frequently in config
17  - mgl_logger - [complete] - a simple logger with support for different logging levels. Support for loggin in a separate thread.
18  - mgl_resource - [complete] - a resource manager "class" that allows the automatic tracking of use and cleaning up of any type of "resource" such as image or audio files or in-project constructs such as entities and windows.
19  - mgl_config - [complete] - a config file parser and resource manager. Will support xml, json and yaml when complete. Converts config files to MglDict.
20  - mgl_json_parse - [complete] - translation from libjansson json_t to mgl_dict
21  - mgl_yaml_parse - [complete] - yaml parser using libyaml, parses it to mgl_dict
22  - mgl_xml_parse - [in progress] - xml parser using expat, parses it to mgl_dict
23 
24 <H3>MoGUL Graphics</H3>
25 These will be the core graphics libraries of MoGUL. This will contain resource managers for resources typically used in game programming.
26  - mgl_graphics - [complete] - initializes and configures rendering pipleline and provides an interface to do basic drawing and frame timing.
27  - mgl_draw - [complete] - provides drawing of shape primitives and integration with Chipmunk physics engine.
28  - mgl_sprite - [complete] - provides a resource manager for animated sprites. Sprites can be setup through sprite sheets or through image sequences.
29  - mgl_svg - provides a resource manager for scalable vector graphics. SVG files will be parsed into MglDict and used to draw.
30  - mgl_actor - [complete] - provides a context for timing sprite/drawing animations, setting animation sequences and triggering events times with key frames.
31  - mgl_font - [complete] - provides a resource manager for loading and displaying fonts / support for word wrapping and font effects.
32 
33 <H3>Extensions</H3>
34  - mgl_input - provides a platform to customize user controls and handle control events.
35  - mgl_combo - provides support to parsing and catching input chains for complex inputs.
36  - mgl_audio - [complete] - provides a resource manager for playing audio clips and music. Wraps SDL_Mixer.
37  - mgl_sound - provides support to load and play sound chunks. Support for channel groups
38  - mgl_music - provides support to load and play music files. Support for playlists
39  - mgl_entity - provides a resource manager for in-game entities. Can be extended with user data and user defined function pointers.
40  - mgl_level - provides a common platform for developing game levels. Will support a range of level styles from tiled to free form as well as multiple layers and parallax effects.
41  - mgl_parallax - provides logic to render parallax layered backgrounds
42  - mgl_tiles - provides a set of functions for loading and dealing with tile maps. Provides the framework for making more robust tiles (animated, active). Will work with the mgl_path seamlessly.
43  - mgl_path - provides a threaded path finding system. Works with tiles, platforms and free form levels.
44  - mgl_gui - provides a graphical user interface platform including resource managers for windows and widgets.
45  - mgl_window - provides the resource manager for windows
46  - mgl_component - provides the resource manager for building window component widgets
47  - mgl_predict - provides a pattern predition implementation. Useful for AI prediction of player actions.
